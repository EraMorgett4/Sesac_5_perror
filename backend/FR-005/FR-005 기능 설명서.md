## 🎯 개요

### 주요 기능
- ✅ **실제 CSV 공사현황 데이터** 로드 및 분석
- ✅ **사용자 위치 기반** 위험도 평가

```
필수 컬럼: 관리코드, 지오코딩주소, 착공일, 준공일, 공사상태, 위도, 경도
선택 컬럼: 좌표상태 (데이터 품질 확인용)
인코딩: UTF-8
```

## 🎛️ 핵심 기능

### 데이터 분석 엔진
- **거리 계산**: Haversine 공식으로 정확한 거리 측정
- **위험도 평가**: 거리별 3단계 (높음/중간/낮음)
- **실시간 필터링**: 진행중인 공사만 자동 추출

### 음성 출력 엔진
- **즉시 재생**: 파일 저장 없이 스피커 직접 출력
- **한국어 최적화**: ko-KR-SunHiNeural 음성 사용
- **설정 가능**: 속도, 톤, SSML 지원

## 📚 함수 사용법

### 1. 메인 서비스 함수

#### `fr005_voice_guidance_service()`
**전체 음성 안내 서비스 실행**

```python
# 기본 사용
result = fr005_voice_guidance_service({
    'lat': 37.5079, 
    'lng': 127.0035
})

# 상세 설정
result = fr005_voice_guidance_service(
    user_location={'lat': 37.5079, 'lng': 127.0035},
    search_radius=1000,  # 검색 반경 (미터)
    data_file_path='custom/path/to/data.csv'  # 특정 파일 지정
)
```

**반환값:**
```python
{
    'user_location': {'lat': 37.5079, 'lng': 127.0035},
    'search_radius': 500,
    'total_constructions_in_data': 30,
    'constructions_found': 2,
    'message': '주변에 2곳의 공사가 진행중입니다...',
    'voice_output': True,  # 음성 출력 성공 여부
    'constructions_details': [...]  # 상세 공사 정보
}
```

### 2. 데이터 로딩 함수

#### `load_real_construction_data()`
**실제 CSV 데이터 로드**


### 3. TTS 서비스 클래스

#### `SimpleTTSService`
**Azure TTS 서비스 관리**

```python
# 서비스 초기화
tts = SimpleTTSService()

# 기본 음성 출력
tts.speak_text("주의! 공사 구역입니다.")

# 고급 설정 (SSML)
tts.speak_with_ssml(
    text="긴급 안내입니다.", 
    rate="1.3",    # 속도 (0.5~2.0)
    pitch="high"   # 톤 (x-low, low, medium, high, x-high)
)
```

## 🔧 문제 해결

### 자주 발생하는 문제

#### 1. 음성이 들리지 않는 경우
```
📝 체크리스트:
□ 시스템 볼륨 확인
□ 스피커/헤드폰 연결 상태
□ Azure Speech Key 설정 (.env 파일)
□ 다른 오디오 프로그램과의 충돌
```

**해결 방법:**
```python
# 1. TTS 서비스 상태 확인
tts = SimpleTTSService()
# 출력에서 "✅ 설정됨" 확인

# 2. 간단한 테스트
result = tts.speak_text("테스트 음성입니다.")
print(result)  # success: True 확인
```

#### 2. 공사가 발견되지 않는 경우
```
💡 해결 방법:
1. 검색 반경 늘리기 (search_radius=1000)
2. 실제 데이터 위치로 테스트 (잠원동, 서초동 등)
3. 데이터 파일 경로 확인
```

#### 3. 파일을 찾을 수 없는 경우
```python
# 절대 경로로 테스트
result = fr005_voice_guidance_service(
    user_location={'lat': 37.5079, 'lng': 127.0035},
    data_file_path='/absolute/path/to/data.csv'
)

# 또는 파일 존재 여부 확인
import os
file_path = "data/processed/road_excavation_processed_20250623_143840.csv"
print(f"파일 존재: {os.path.exists(file_path)}")
```

### 오류 메시지 해석

| 메시지 | 원인 | 해결책 |
|--------|------|--------|
| `Azure SDK 없음` | 패키지 미설치 | `pip install azure-cognitiveservices-speech` |
| `API Key 없음` | 환경변수 미설정 | `.env` 파일에 `AZURE_SPEECH_KEY` 추가 |
| `파일을 찾을 수 없음` | 데이터 파일 없음 | 파일 경로 확인 또는 샘플 데이터 사용 |
| `TTS 실패` | 네트워크/API 오류 | 인터넷 연결 및 Azure 서비스 상태 확인 |
